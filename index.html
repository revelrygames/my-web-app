Это веб-приложение в Telegram с красивым фоном.

/* Добавляем фоновое изображение на весь экран */ body { background-image: url('cat'); /* Путь к твоему изображению */ background-size: cover; /* Изображение будет покрывать весь экран */ background-position: center; /* Центрируем изображение */ background-repeat: no-repeat; /* Изображение не будет повторяться */ height: 100vh; /* Высота экрана */ margin: 0; /* Убираем отступы */ } /* Стили для основного контента */ .background { text-align: center; color: white; padding-top: 20%; }
